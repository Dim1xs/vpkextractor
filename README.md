# vpkextractor
Python script for extracring VPK (Valve Package) format files. <br>
**Termux on Android tested only.**

# Installation
**Termux**
```
pkg upgrade
pkg install git -y 
pkg install python -y 
git clone https://github.com/Dim1xs/vpkextractor.git
cd vpkextractor
pip3 install -r requirements.txt
```

# Using
```
cd vpkextractor
python extractor.py
```
